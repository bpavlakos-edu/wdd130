#1. Telling a Computer What To Do

Programming is a very interesting concept, essentially at it's core it's telling a computer what to do, the only issue is that it will do exactly what you tell it to do! It's a fantastic skill, that can help you make anything from video games, to budgeting software, and even bots that surf the web to help get you camping reservations!

But to a beginner, programming can be very confusing, so confusing you don't know where to start! I felt that way too, when I first began programming at the age of 12. Luckily, unlike me at age 12, today you'll be learning how to program in an established programming language, one that was meant to be programmed in using an IDE (which is computer software that helps you program and checks for errors), not an undocumented scripting language from a 1999 video game, that I programmed in using notepad.

So to begin you on your programming journey, today you'll be learning one of the first programming languages I formally learned: Processing!

What's processing? Well, you'll find out in the next section.

#2. Intro to Processing

Processing is a Java Based programming langauge, created for a unique purpose.

What makes it so special?

As stated on it's overview page:

From the beginning, Processing was designed to be as simple as possible for beginners... The same elements taught in a beginning high school or university computer science class are taught through Processing, but with a different emphasis. With its focus on creating visual, interactive media, students new to programming find it satisfying to make something appear on their screen within moments of using the software.

From that we know that Processing is a great langauge to start on, but as an experienced Java programmer, I can tell you that everything you learn here will help you to learn a language like java too! The overlap between the two is huge, it's just processing has prebuilt code to make certain things take less time to type in.

On the flip side, Processing has some real world application too! It's the primary language for programming Arduino Microcontrollers, which are basically a programmable electron chip, that can control or interact with electonic devices and parts. But you'll learn about that more in the next lesson!

Just know that what you learn about programming here will carry on to countless things in the future!

Sounds great right? Yes it does! Go download Processing now here on their website so we can continue your computer programming journey!
