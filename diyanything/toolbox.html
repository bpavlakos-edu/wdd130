#Tools For Computer Programming

Processing

An open source Java based programming langauge, made for beginners!

I learned it in high school, and the things I learned in it I still use today!

The best part about it, is not only that it's simple to use, but it's also free.

There are several other versions of it too, designed to use different langauges too like Python! But the Java Version, which is also the main version is the one I'm most familiar with!

You can download it on their home page here

Intelli-J for Java

The best IDE for Java by a landslide, it's abilities and features are unparalleled!

The best part about it, is that it's community version is absolutely free for personal use. So if you're making your own Java projects, there's really no good reason not to use it!

You can download it on their home page here

There are other free IDEs on that website too, like pyCharm for Python, but there are also paid IDEs and Extensions too.

Visual Studio Code

The swiss army knife of computer programming tools, it can help you program in just about any programming language as long as you have it installed, and find (or make) an extension for it!

It supports the popular web design langauges: HTML, CSS, and VSCode. It also supports Java, Python, C#, C++, LUA, and many more languages!

Even if it doesn't support the langauge you're using (like some of the modding langauges I have), you can still use it to have access to it's amazing features, like multi-line cursors, multiple workspaces, quick navigation, and live sharing over the internet!

It's also absolutely free to download, so why not give it a chance!

Notepad++

Free Code-Focused Notepad

Do you wish there was a version of notepad designed for coding? Well, here's your dream come true!

Notepad++ is a text editor designed for coding, it has the ability to search across files, collapse code blocks, and has autocompletion!

It supports many langauges immediately. But one of it's most powerful features, is the ability to create "user-defined langauges" or UDLs for short. UDLs let you define the syntax for a specific file type, allowing you to take full advantage of the features recognzied languages have. It's really great for obscure or undocumented scripting languages!

Notepad++ also has extensions, allowing for even further customization!

You can download it on their page here

Ghidra

Created by the National Security Agency, Ghidra is a dissasembly tool that was released to the public in 2017. In other words, it is software that can analyze an exe and turn it into somewhat comprehendable code.

It is completely free, open source, community-driven, cutsomizable, and extremely powerful. And the feature list will only grow as time goes on!

I've used this program to help me reverse engineer many games and software, by labeling unknown functions, and taking advantage of ghidra's features (like string tables, hex search, and tree view). And it has taught me how to look at assembly and see how it translates into code!

The only real catch here, is that you need to have a few gigabytes of space to store the ghidra projects! Though, in my entire time reverse engineering a grand total of about 28 different programs I've only used up 2.3gb.

You can download it, and it's source code on their page here or directly on their github page here.

Malwarebytes' free version has always been reliable to me, and it's definitely better than nothing!

Best part is that it's completely free, and all online.

Gimp

Gimp is a raster photo editor comparable in capability to photoshop.

Although it might not have as many features as photoshop does, it's completely open source, and completely free.

Don't be fooled by it's lack of a price tag, it's an extremely powerful photo editor, that I've used for years, and it's only going to get better as time goes on, as it's next update is geared towards non-destructive editing like photoshop currently has.

The hero photos for each lesson page, was created using gimp, it's filters, and the fractal explorer.

Not only is it good for graphic design, it's also good for reverse engineering since it supports importing "Raw Image Data" from bytes!

M.U.G.E.N.

Customizable and Free 2D Fighting Game

This is a 2D fighting game unlike any other. Sure using Kung-Fu man is fun, but the real fun begins when you download characters from online!

There are 1206 different characters on the MUGEN wiki alone, and there are definitely more websites to get things from too!

But that's not the end of it, you can download stages, new interfaces, and even entire MUGEN packages. The possibilities are endless (as long as you avoid sketchy websites and don't download offensive content)!

You can download it from one of the links on it's MUGEN wiki page, you can also download characters here too!

Also you can make your own chacters using Fighter Factory, and add your newly downloaded characters easily with VSelect

The screenpack I use is International Mugen Tournament, which has a layout that supports adding up to 8400 characters! Though personally, I've only used the 336 character version!

I.K.E.M.E.N. GO

Open Source Engine based on M.U.G.E.N.

Like M.U.G.E.N., but you want to literally change the source code? Well look no further, I.K.E.M.E.N. is the Open Source project for you!

It does everything M.U.G.E.N. does without being trapped in an EXE that won't be updated ever again! Not to mention that it already has some enhancments over regular M.U.G.E.N. too, including net-play!

IKEMEN GO is the latest version of IKEMEN written in the Google Go language, it has a lot of enhancments over I.K.E.M.E.N. itself, and is being actively maintained!

You can download I.K.E.M.E.N. GO on it's github page here, and the original I.K.E.M.E.N. on it's OSDN page here, or read more about it here on the MUGEN wiki!
